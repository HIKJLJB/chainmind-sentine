# ChainMind Sentinel

AI-powered Web3 Automation Agent.

ChainMind Sentinel is an intelligent blockchain task execution framework designed for:

- Airdrop automation
- Epoch monitoring
- Auto claim
- AI-driven task execution
- Wallet interaction automation
- Multi-chain monitoring

## Features

- AI Agent Decision Engine
- Automatic Wallet Interaction
- Epoch Window Detection
- Reward/Risk Analysis
- Telegram Notification
- Multi-chain Support

## Installation

```bash
npm install
```

## Start

```bash
node index.js
```
